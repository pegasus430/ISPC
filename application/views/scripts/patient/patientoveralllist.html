Erluterung zur Darstellung der Spalten in der Patientenliste

Sie knnen mit dieser Liste komfortabel steuern, welche Inhalte in der Tabelle fr Sie angezeigt werden sollen. Dabei gibt es drei Zustnde. Diese sind in der Tabelle unten erlutert.
Inaktiv - Diese Spalte wird in der Tabelle nicht angezeigt
Aktiv - Diese Spalte wird in der Tabelle in dem Aufklappteil angezeigt. Um diese Spalte zusehen, mssen Sie am rechten Rand den Pfeil klicken.
Diese Spalte wird direkt in der Liste angezeigt. Es knnen bis zu 6 Spalten in diesem Status sein und damit hervorgehoben angezeigt werden. Wenn Sie eine 7. Spalte versuchen so auszuwhlen, springt der Schalter wieder auf normal aktiv um. Sie mssen in diesem Fall erst eine Spalte aus diesem Zustand entfernen, um eine neue Spalte auszuwhlen.
